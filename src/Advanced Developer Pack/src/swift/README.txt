Dependencies:

	To install dependencies, first install cocoapods (sudo gem install cocoapods)
	Open the project folder in terminal (the folder which contains the file 'Podfile') and type "pod install"
	Open the newly created xcode workspace and the project should build.
	
	For more help on cocoapods, please view this guide: https://guides.cocoapods.org/using/getting-started.html
